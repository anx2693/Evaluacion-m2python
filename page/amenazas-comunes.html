Phishing

Es un tipo de ataque de ingeniería social donde los atacantes se hacen pasar por entidades legítimas (como bancos, redes sociales, etc.) para engañar a las víctimas y obtener información sensible como contraseñas, datos bancarios, etc. Suelen utilizar correos electrónicos, mensajes de texto o sitios web falsos para lograr su objetivo.

Ransomware

Es un tipo de malware que cifra los datos de la víctima, impidiendo su acceso, y exige un rescate a cambio de la clave de descifrado. Los ataques de ransomware pueden ser devastadores para individuos y organizaciones, causando pérdida de datos y interrupción de operaciones.

Malware

Es un término general que engloba cualquier tipo de software malicioso diseñado para dañar, infiltrarse o tomar control de sistemas informáticos. Incluye virus, gusanos, troyanos, spyware, adware, entre otros. El objetivo del malware puede variar, desde robar información hasta dañar el sistema o simplemente molestar al usuario.

Ataques DDoS

Un ataque DDoS inunda un sistema o red con tráfico masivo, proveniente de múltiples fuentes, con el objetivo de sobrecargar sus recursos y hacer que deje de funcionar para usuarios legítimos. Estos ataques pueden ser muy efectivos para interrumpir servicios en línea, como sitios web, servidores o servicios de red.
